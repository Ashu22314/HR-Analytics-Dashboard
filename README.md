# HR-Analytics-Dashboard


## 📌 Overview

This project showcases an interactive **Power BI Dashboard** designed to analyze and visualize key business data. The dashboard provides meaningful insights through dynamic charts, KPIs, and filters to support data-driven decision-making.

---

## 🚀 Features

* 📈 Interactive dashboards with real-time filtering
* 📊 Visual representation of key performance indicators (KPIs)
* 🔍 Drill-down analysis for deeper insights
* 📅 Time-based trend analysis
* 📌 User-friendly and visually appealing layout

---

## 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel / CSV (Data Source)
* Data Cleaning & Transformation (Power Query)
* Data Modeling (Relationships, DAX)

---

## 📂 Dataset

The dataset used in this project includes:

* Customer data
* Transaction details
* Sales / Revenue information
